# Bus Ticket Booking Website

## Overview
Developed a comprehensive bus ticket booking website featuring user-friendly interfaces and backend functionality for managing bookings, routs and payments.

## Key Features
- **Front-End**: Utilized HTML, CSS, and JavaScript to create a responsive and visually appealing user interface.
- **Back-End**: Implemented server-side logic and database connections to handle bookings, manage routes, and process payments.
- **Payment Integration**: Included options for various payment methods and a ticket download feature with notifications.

## Technology Stack
- **Front-End**: HTML, CSS, JavaScript
- **Back-End**: Server-side scripting-script.js
- **Database**: MySQL

## Objective
To deliver a fully functional bus ticket booking system that enhances user experience and streamlines ticket management processes.
